ifx_Error_t ifx_window_init_chebyshev ( ifx_Vector_R_t *  win,

)
static

This function generates the coefficients of a Chebyshev window for a given length.

Parameters
[in,out]winPointer to an allocated and populated, but empty vector instance defined by ifx_Vector_R_t to be filled with Chebyshev Window coefficients.
[in]at_dBDefines the attenuation in dBs required to generate the pass band ripple for a Chebyshev window. This must be a positive number.

Reference: Dolph Chebyshev window according to http://practicalcryptography.com/miscellaneous/ machine-learning/implementing-dolph-chebyshev-window/
