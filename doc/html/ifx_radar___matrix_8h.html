ifx_Error_t ifx_matrix_add_c ( const ifx_Matrix_C_t *  matrix_l,
const ifx_Matrix_C_t *  matrix_r,
ifx_Matrix_C_t *  result 
)

Adds two complex matrices and saves the result of addition into given output matrix.

This function supports in-place addition operation, therefore it is allowed one of the input and output matrix pointing to the same matrix.

Math operation implemented by this method: Z = X + Y

Parameters
[in]matrix_lComplex matrix as left argument (X in above equation).
[in]matrix_rComplex matrix as right argument (Y in above equation).
[out]resultComplex matrix where the result of addition will be stored (Z in above equation). It must be allocated and of the same dimensions as the given input matrix. Can be in-place.
Returns
ifx_Error_t Error if dimensions do not match or NULL is passed as matrix pointer.
