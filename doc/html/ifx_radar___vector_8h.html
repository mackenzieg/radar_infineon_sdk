Performs rotation on a complex vector elements by factor and elements shifted out are fed back to the vector like a circular ring.

Input and output memories should be of same type and size, else error code is returned.

Parameters
[in]inputComplex array on which shift operation needs to be performed
[in]factorRotation factor (+ive for rotate clockwise, -ive not supported yet), elements shifted by this factor in a circular ring manner
[out]outputRotated vector by a given rotation factor
