◆ ifx_mem_get_total_alloc_size()

int ifx_mem_get_total_alloc_size ( )

This function returns the number of the total allocated memory in bytes.

Internal tracking must be enabled with TRACK_MEMORY_ALLOCATION defined to enable this function.

Returns
Total number of bytes allocated. If tracking is not enabled this function returns -1.
