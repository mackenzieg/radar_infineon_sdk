Detailed Description

This file defines the API of mathematical operations such as mean calculation, scalar removal from an array, element wise addition, subtraction and multiplication of arrays, scalar multiplication of an array, absolute value calculation, maximum value in an array.

Computes element wise addition of two complex vectors and stores it in result vector.

This function allows in-place calculation, that means a input vector can also be the output vector at the same time if the input vector can be overwritten.

Parameters

[out]resultPointer to output vector the sum is stored to. Can be one of the input vectors v1 or v2.
