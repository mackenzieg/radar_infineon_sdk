Field Documentation

◆ fft_input_len

uint32_t fft_input_len

Size of the input (time domain) data samples, in units of sample count.

Should be non-zero and not greater than FFT size.

◆ fft_size

ifx_FFT_Size_t fft_size

The FFT parameters are defined by structure ifx_FFT_Size_t.

Must be power of 2.

◆ fft_type

ifx_FFT_Type_t fft_type

FFT type is defined by ifx_FFT_Type_t.

Type is defined based on FFT input data nature, i.e. real or complex.

◆ input_sampling_freq_khz

uint32_t input_sampling_freq_khz

This value is used to calculate the frequency bins in the FFT output.

◆ is_normalized_window

uint8_t is_normalized_window

Setting to zero will turn off this feature.

If set to non-zero value the effect of different window type is normalized. Changing window type will result in similar magnitude range.

◆ mean_removal_flag

uint8_t mean_removal_flag

Zero value considered as False (skip mean removal feature), while non-zero value would be considered as true for mean removal.

◆ window_config

ifx_Window_Config_t window_config

Window type, length and attenuation used for range FFT.
