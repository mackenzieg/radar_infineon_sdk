This function instantiates a handle to an MTI filter structure ifx_MTI_s which contains information required to perform MTI (Moving Target Indication) filtering.

The MTI filter is basically a first order IIR (Infinite Impulse Response) filter.

The MTI filter is typically used on a range spectrum to filter out targets that are stationary. Thus the name "Moving Target Indication" filtering. The effect of static target removal with the MTI filter can be seen when applied on the range spectrums from successive FMCW frames. The ifx_MTI_s stores a historical component of range spectrums which have information of static targets in previous spectrums. This historical component is subtracted from the range spectrum of the next incoming FMCW frame.

The filter co-efficient \( alpha \) needs to be specified when creating the an instance of the MTI module.

The MTI filtering is realized according to the following formula:

\[ RSF_n = |RS_n - RSH_{n-1}| \\ RSH_n = alpha*RS_n + (1 - alpha)*RSH_{n-1} \]

Where:

  • \( RS_n \) = current ( \( n^{th} \)) input vector
  • \( RSF_n \) = current ( \( n^{th} \)) MTI filtered output vector
  • \( RSH_{n-1} \) = Historical component from the previous \( n-1 \) input vector filterings
  • \( RSH_n \) = Historical component after filtering the current ( \( n^{th} \)) input vector
  • \( alpha \) = Filter-coefficient of the IIR filter used for MTI
